# 🎙️ Punjabi Speech Recognition using XLSR

An end-to-end Automatic Speech Recognition (ASR) system for Punjabi using Facebook AI’s XLSR (Cross-Lingual Speech Representation) model. This project focuses on improving transcription accuracy for low-resource languages through fine-tuning and preprocessing techniques.

---

## 🚀 Features
- Fine-tuned XLSR model for Punjabi speech recognition
- Handles low-resource language challenges
- Audio preprocessing and feature extraction
- Model evaluation using Word Error Rate (WER)
- Scalable pipeline for multilingual ASR tasks

---

## 🛠️ Tech Stack
- Python
- PyTorch
- HuggingFace Transformers
- Librosa / SoundFile
- NumPy, Pandas

---

## 📂 Project Workflow
1. Data Collection (Punjabi speech dataset)
2. Audio preprocessing (resampling, normalization)
3. Feature extraction
4. Model fine-tuning (XLSR)
5. Evaluation using WER
6. Inference on new audio samples

---

## 📊 Results
- Improved transcription accuracy on Punjabi dataset
- Effective adaptation of multilingual pretrained model
